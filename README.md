- 캡스톤 디자인 및 팀 프로젝트를 하는 학생들을 위한 웹 서비스

- 채팅 기능
- 회의록 작성 기능
    - 회의록 작성, 수정, 삭제
    - 동시 편집
    - 노션과 비슷한 기능 제공
